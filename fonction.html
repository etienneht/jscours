Les Fonctions

Une fonction est un bloc de code réutilisable qui effectue une tâche précise

Elle permet d'organiser ton code et d'éviter les répétitions et rend ton code plus lisible

            
                function nomDeLaFonction() {
                    //instruction à executer
                }

Comment les range-t-on dans le code ?

Les fonctions principales sont rangées plus vers le haut que les fonctions "utilitares"

            

                const panier = [
                { name: "Pomme", price: 2, quantity: 5},
                { name: "Kiwi", price: 1, quantity: 50},
                { name: "Mangue", price: 9, quantity: 25},
                ]


                // fonction qui additionne 2 nombres

                // c'est une fonction utilitaire

                function add(nbrA, nbrB){
                return nbrA + nbrB;
                }

                // autre fonction utilitaire

                function multiply(nbrA, nbrB){
                return nbrA * nbrB;
                }

                // fonction principale qui va calculer le prix total du panier

                function calculatePriceCart(cart){
                // avant de calculer, on va initialiser le total
                let total = 0;

                for (let i = 0; i < cart.length; i++){
                // pour faciliter la lecture, on renomme l'élément actuel de la boucle par une variable plus simple à écrire
                const item = cart[i];
                // le retour de la fonction multiply "injecte" son résultat dans itemPriceTotal
                const itemPriceTotal = multiply(item.price, item.quantity);
                total = add(total, itemPriceTotal)
                }
                return total
                }

                console.log(calculatePriceCart(panier))

Les Arguments

Un argument est une valeur qu'on donne a une fonction quand on l'appelle pour qu'elle travaille avec cette valeur

  • Quand on déclare la fonction on parle de paramètres
  • Quand on appelle la fonction on parle d'argument
            

                function bonjour(prénom) {
                    console.log("Bonjour" + prenom)
                }

                bonjour("Jean-Michel");

Plusieurs arguments

Une fonction peut avoir autant de paramètres que vous voulez, et tu dois passer le même nombre d'argument quand tu l'appelles

Si le nombre d'argument est en trop ou qu'il en manque, la fonction retourne undefined

            

                function add(nbrA, nbrB){
                return nbrA + nbrB;
                }

                console.log(add(7, 19)); // 26

Valeurs par défaut

                

                    function bonjour(prénom = "Sebastien") {
                    console.log("Bonjour" + prenom)
                    }

                    bonjour(); // "Bonjour Sebastien"
