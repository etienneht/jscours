1. Introduction à JavaScript

C'est quoi le JavaScript ?

Le JavaScript (JS) est un langage de programmation qui donne de l'intéractivité aux pages web.

  • HTML = Structure ou squelette
  • CSS = Style (les couleurs, la mise en page, etc...)
  • JS = Logique (les réaction à un utilisateur, des calculs et la dynamique de la page)

Où s'éxécute le JS ?

  • Dans le navigateur (Chrome, Firefox, Safari, etc...)
  • Sur un serveur qui tourne avec Node.js -> il ne sera plus Front-End mais Back-End comme le PHP ou le Python etc...

À retenir : même langage, mais deux environnement différents.

Premier pas : console, balise script

  1. Ouvrir votre navigateur préféré
  2. Click droit -> inspecter, ou si vous êtes sur Windows F12.
  3. Aller dans l'onglet console
