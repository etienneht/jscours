Les bases du langage JavaScript

Les variables

Une variable est un espace mémoire qui stocke une valeur (un nombre, du texte, etc...)

En JS, on déclare une variable avec les mots-clés let, const et anciennement var (toujours utilisé dans du vieux code, par contre ne plus l'utilisé)

Les boucles

Une boucle est une structure qui permet de répéter un bloc de code tant qu'une condition est vraie.

C'est une façon automatique de répéter des instructions sans devoir les écrire plusieurs fois

Les types de boucles

  • La boucle While
  • La boucle Do...While
  • La boucle For
  • La boucle For...Of
  • La boucle For...in

La boucle While

On l'utilise quand on ne connait pas à l'avance le nombre de tours

            
                //Condition => doit être vrai pour pourvoir continuer la boucle.
                //(comme le if, cependant ne jamais mettre de "==" ou de "===")
                while (condition) {
                    // code à exécuter
                }

Exemple concret

            
                let n = 0;
                while (n < 6) {
                    console.log(n + " * 5 est égale à " + n * 5);
                    n++;
                }

La boucle Do...While

C'est comme la boucle while, mais elle s'exécute au moins une fois avant de tester la condition

            
                do {
                    // 
                } while (condition);

Exemple concret

            
                let password;
                do {
                    // 
                } while (condition);

La boucle For

La plus classique et la plus utilisée quand on connait à l'avance le nombre de répétition

            
                for (initialisation; condition; incrémentation) {

                }

Exemple concret

            
                for (let i = 0; i < 5; i++) {
                    console.log("Compteur : " + i);
                }

Une boucle for permet de traverser un tableau à partir de ses index

            
                let voitures = ["Dacia", "Lada", "Audi", "Daihatsu", "Toyota", "Chevrolet"];

                for (let i = 0; i < voitures.length; i++) {
                    console.log(voitures[i]);
                }

For...of

                for (let i = 0; i < voitures.length; i++) {

Les mots clés BREAK et CONTINUE

break permet de sortir immédiatement d'une boucle

            

                //break
                for (let i = 0; i < 10; i++){
                    if (i === 5) {
                        break;
                    }
                    console.log(i)
                }

Le continue permet de sauter une iteration sans arrêter la boucle

                continue;
                }
                console.log(i)

LES TABLEAUX

Un tableau en JavaScript est un objet spécial qui peut contenir plusieurs valeurs dans une seule variable

            
                const fruits = ("pomme", "banane", "orange")

Comment accéder aux éléments ?

            
                const fruits = ["pomme", "banane", "orange"]

                console.log(fruits[0]); "pomme"
                console.log(fruits[1]); "banane"
                console.log(fruits[2]); "orange"

Modifier un élément du tableau

            
                const fruits = ["pomme", "banane", "orange"]

                fruits[1] = "kiwi";

Ajouter et supprimer des éléments dans un tableau

            
                const fruits = ["pomme", "banane", "orange"]

                const fruits2 = ["pomme", "banane", "orange"]

                //ajouter un élément à la fin du tableau
                fruits2.push("fraise");
                //ajouter un élément au début
                fruits2.unshift("melon");
                //supprimer le dernier élément 
                fruits2.pop();
                //supprimer le premier élément 
                fruits2.shift()

Autres fonctions utiles

            
                //autres fonctions utiles

                const fruits3 = ["framboise", "avocat", "coing", "poire"];

                //trouver l'index d'un élément
                let poseFruit = fruits3.indexOf("tomate")
                console.log(poseFruit)
                //verifier si un élément est présent
                console.log(fruits3.includes("orange"))
                //convertir un tableau en chaine de caractères
                let saladeDeFruit = fruits3.join(", ")
                console.log(saladeDeFruit)
