Set Timeout() et SetInterval()

SetTimeout()

Ca execute une fonction une seule fois après un certain temps

            
                
                    setTimeout(function (){
                    console.log("3 secondes se sont écoulées !")

                }, 3000)

SetInterval()

Execute une fonction de manière répétée à interval régulier

            
               
                    setInterval(function (){
                    console.log("une seconde vient de passer !")

                }, 1000)

ClearTimeout() et ClearInterval()

Pour pouvoir supprimer vos timers il faudra les mettre dans une variable

            
               
                const timer = setTimeout(function (){
                console.log("3 secondes se sont écoulées !")

            }, 3000)

                clearTimeout(timer) // c'est plus interressant dans un évènement

                const interval = setInterval(function (){
                console.log("une seconde vient de passer !")

            }, 1000)

                clearInterval(interval)
